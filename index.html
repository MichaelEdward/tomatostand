Tomatostand

Chrome Extension to show Verizon's TV Guide sorted by Rotten Tomato score

Download this project as a .zip file Download this project as a tar.gz file

This extension will add a red Tomato Stand Listing link to the top of Verizon's TV Listing. Once the new link is clicked, it will load every movie for the specified number of future days for the specified number of channels and sort them by Rotten Tomato Score. A green score indicates that there was no critic score found, and is showing the audience score instead.

Next to each movie description is also a red link for each showing. Clicking that link will program your DVR to record that showing.

At the bottom of the page is a Mark All Read link which will gray out the movie the next time the listing is shown.

To prevent hammering Verizon's servers, fetching of listings is throttled and could take several minutes.
